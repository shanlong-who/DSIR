# List SDG Indicators

Fetches the list of SDG indicators from the UN SDG API.

## Usage

``` r
sdg_indicators()
```

## Value

A list (or [tibble](https://tibble.tidyverse.org/reference/tibble.html))
of SDG indicators, or `NULL` when the service is unreachable.

## See also

[`sdg_targets()`](https://shanlong-who.github.io/DSIR/reference/sdg_targets.md),
[`sdg_data()`](https://shanlong-who.github.io/DSIR/reference/sdg_data.md).

## Examples

``` r
if (FALSE) { # \dontrun{
sdg_indicators()
} # }
```
